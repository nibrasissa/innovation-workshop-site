# ورشة الابتكار العملية – موقع تعريفي

موقع ثابت (Static) باللغة العربية لورشة من 8:30 ص إلى 2:30 م. يتضمن روابط **التسجيل** و**التقييم**، والجدول، والأهداف، والتفاصيل.

## النشر على GitHub Pages (تلقائي)
1) أنشئ مستودعًا جديدًا على GitHub (عام).
2) ادفع هذا المشروع إلى الفرع `main`.
3) من تبويب **Actions** سيعمل سير العمل تلقائيًا وينشر الموقع.
4) عنوان الموقع سيكون: `https://<user>.github.io/<repo>/`.

### أو: تفعيل Pages من الإعدادات
Settings → Pages → Build & deployment → Source: *Deploy from a branch*، Branch: `main`، Folder: `/ (root)`.

## تطوير محلي
افتح `index.html` مباشرة في المتصفح. لا يعتمد المشروع على أي بناء (no build).
